1. Для просмотра работы необходимо открыть файл index.html в браузере (Ffirefox). Данное приложение не предназначено для браузера Chrome, так как загружает локальные файлы при помощи xml http request, что противоречит политике безопасности Chrome.2. Данная работа представляет собой примеры некоторых готовых частей проекта, но не является полностью завершенным проектом.

3. Страница page-1 реализует следующий функционал:
 - Пролистывание слайдера с использованием навигационных стрелок;
 - Пролистывание слайдера с использованием навигационных маркеров.

4. Страница page-2 реализует следующий функционал:
 - Выбор приложения из списка для просмотра подробной информации;
 - Добавление приложения в корзину;
 - Удаление приложения из корзины;
 - Сохранение данных корзины в local storage после закрытия браузера;
 - Синхронизация данных корзины при помощи local storage при работе с приложением на нескольких вкладках одновременно;
 - Реализация оплаты.
