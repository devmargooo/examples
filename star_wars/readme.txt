1. Для просмотра работы необходимо открыть файл index.html в браузере (FF, Safari). Данное приложение не предназначено для браузера Chrome, так как загружает локальные файлы при помощи xml http request, что противоречит политике безопасности Chrome.2. Функционал данного приложения включает следующие возможности:• Фильтрация карточек перснажей по половому признаку;• Просмотр подробной карточки персонажа.
